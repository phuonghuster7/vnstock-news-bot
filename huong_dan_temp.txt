Step 5: converter.py — DataFrame → Qlib Binary
Spec: File quan trọng nhất. Phải align float32 chính xác với calendar.
Qlib binary format:
File .day.bin = float32 array (little-endian)
Length = số ngày trong day.txt
Ngày không có data = np.nan
Phải align 1-1 với index của calendar
def df_to_qlib_bin(df: pd.DataFrame, calendar: list[str], col: str) -> np.ndarray:
    """Tạo float32 array aligned với calendar"""
    cal_index = {d: i for i, d in enumerate(calendar)}
    arr = np.full(len(calendar), np.nan, dtype=np.float32)
    for _, row in df.iterrows():
        idx = cal_index.get(str(row['date']))
        if idx is not None:
            arr[idx] = float(row[col])
    return arr

def write_bin(arr: np.ndarray, path: str):
    arr.astype('<f').tofile(path)   # little-endian float32, bắt buộc
Features tạo cho mỗi symbol:
open.day.bin    close.day.bin    high.day.bin
low.day.bin     volume.day.bin   factor.day.bin (= 1.0 toàn bộ)
vwap.day.bin    (= (high+low+close)/3)
