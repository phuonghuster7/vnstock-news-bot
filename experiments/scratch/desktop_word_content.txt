Hướng đi thực sự: Feature Engineering, không phải tuning thêmHyperparameter tuning đã làm xong và đã overfit. Bước tiếp theo duy nhất có ý nghĩa là thêm signal source mới. Có 3 candidates theo thứ tự khả thi:Candidate 1 — Sector Momentum (khả thi nhất, không cần data mới)VN30 gồm ~8 ngành. Momentum sector-level thường mạnh hơn stock-level vì dòng tiền VN rotate theo ngành rõ ràng hơn theo cổ phiếu riêng lẻ.python# vnstock_qlib/features/sector_momentum.pySECTOR_MAP = {    "VCB": "bank", "BID": "bank", "CTG": "bank", "MBB": "bank",    "TCB": "bank", "VPB": "bank", "HDB": "bank", "STB": "bank",    "VHM": "realestate", "VIC": "realestate", "NVL": "realestate",    "HPG": "steel",  "HSG": "steel",    "VNM": "consumer", "MSN": "consumer", "SAB": "consumer",    "FPT": "tech",   "CMG": "tech",    "GAS": "energy", "PLX": "energy", "PVD": "energy",    "VJC": "aviation", "HVN": "aviation",    "MWG": "retail",}def add_sector_momentum_features(df: pd.DataFrame) -> pd.DataFrame:    """    Thêm 2 features:      sector_ret_5d  : return trung bình của sector trong 5 ngày      sector_rel_ret : return của cổ phiếu - return của sector (relative strength)    Cả hai tính cross-sectional theo ngày, không cần data mới.    df: MultiIndex (datetime, instrument), có cột ret_5d    """    df = df.copy()    df["sector"] = df.index.get_level_values("instrument").map(SECTOR_MAP)    # Sector average return mỗi ngày    sector_avg = (        df.groupby(["datetime", "sector"])["ret_5d"]        .transform("mean")    )    df["sector_ret_5d"]  = sector_avg    df["sector_rel_ret"] = df["ret_5d"] - sector_avg  # stock vs sector    df = df.drop(columns=["sector"])    return dfCandidate 2 — Foreign Flow Proxy (cần thêm data nhưng vnstock có)Khối ngoại mua/bán là leading indicator mạnh nhất trên VN market. vnstock có Quote.intraday() với foreign buy/sell volume.python# vnstock_qlib/features/foreign_flow.pydef fetch_foreign_flow(symbol: str, start: str, end: str) -> pd.DataFrame:    """    Lấy net foreign flow từ vnstock.    Returns DataFrame: index=date, columns=[foreign_net_vol, foreign_net_val]        foreign_net_vol > 0 → khối ngoại mua ròng → bullish signal    """    from vnstock import Quote        # vnstock trả về foreign buy/sell trong trading data    df = Quote(symbol=symbol, source="KBS").history(        start=start, end=end, interval="1D"    )        # Columns tùy version vnstock — đọc docs/vnstock/ để confirm tên column    # Thường là: foreignBuyVolume, foreignSellVolume    if "foreignBuyVolume" in df.columns:        df["foreign_net_vol"] = df["foreignBuyVolume"] - df["foreignSellVolume"]        df["foreign_flow_5d"] = df["foreign_net_vol"].rolling(5).sum()        df["foreign_flow_20d"] = df["foreign_net_vol"].rolling(20).sum()                # Normalize: rank cross-sectional (consistent với pipeline hiện tại)        return df[["foreign_net_vol", "foreign_flow_5d", "foreign_flow_20d"]]        return pd.DataFrame()Candidate 3 — Vol-adjusted Momentum (không cần data mới, implement ngay)IC thấp một phần vì momentum signal bị dominated bởi high-vol stocks. Sharpe-weighted momentum ổn định hơn:python# Thêm vào alpha101.pyEXTRA_FEATURES = [    # Vol-adjusted momentum — signal mạnh hơn raw return    ("(Ref($close,5)/$close-1) / (Std(Ref($close,1)/$close-1, 20) + 1e-8)",     "sharpe_momentum_5d"),        ("(Ref($close,20)/$close-1) / (Std(Ref($close,1)/$close-1, 60) + 1e-8)",     "sharpe_momentum_20d"),        # Price acceleration — second derivative của momentum    # Dương = momentum đang tăng tốc, âm = đang giảm tốc    ("(Ref($close,5)/$close-1) - (Ref($close,10)/Ref($close,5)-1)",     "momentum_accel_5d"),        # Liquidity-weighted: volume spike + price up = stronger signal    ("($volume/Mean($volume,20)) * (Ref($close,5)/$close-1)",     "vol_weighted_mom_5d"),]Thứ tự thực hiệnBước 1 — Thêm EXTRA_FEATURES (vol-adjusted momentum)          Không cần data mới, implement trong 1 giờ          Rebuild qlib_data → retrain → đo IC delta                    ↓          IC tăng > 0.005?            Có → tiếp tục   |   Không → skip, sang bước 2                    ↓Bước 2 — Sector momentum features          Dùng SECTOR_MAP + ret_5d có sẵn          Không cần fetch thêm gì                    ↓Bước 3 — Foreign flow (nếu IC sau bước 1+2 vẫn < 0.04)          Cần fetch thêm data, tốn thời gian hơn                    ↓Bước 4 — Walk-forward lại với feature set mới          Target: IC mean > 0.04, ICIR > 0.35 stable qua 3 windowsTarget cụ thể để có thể live trade:MetricHiện tại (V4)Target minimumIC mean0.0289> 0.04ICIR0.1337> 0.35W-F win rate1/3 windows2/3 windows
